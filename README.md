# P_COMPILER
### Techs:
- NextJS
- Python FAST API


### Installations:
- Install node modules:
```bash
  yarn install
 ```
  
- Install python vendor
```bash
python3 -m venv python/venv
source venv/bin/activate 
yarn run pip:install
```

### Run applications:
- In web app environment:
```bash
yarn run next:dev 
```

- In electron environment:
```bash
yarn run dev 
```
